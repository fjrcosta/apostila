Capítulo 14 Análise de Correlação e Regressão

14.1 Regressão Linear Simples

14.1.1 Introdução

A teoria de Regressão teve origem no século XIX com Galton. Em um de seus trabalhos, estudou a relação entre a altura dos pais e dos filhos \((X_i, Y_i)\), procurando saber como a altura do pai influenciava a altura do filho. Notou que se o pai fosse muito alto ou muito baixo, o filho teria uma altura tendendo à média.

Em muitos problemas há duas ou mais variáveis que são relacionadas, conhecendo os efeitos que algumas variáveis exercem sobre outras. Dessa relação pode-se relacioná-las por meio de uma expressão matemática, utilizada para estimar o valor de uma das variáveis quando conhecemos os valores das outras.

Em geral, suponha que haja uma única variável dependente, ou resposta, \(Y\) que depende de \(k\) variáveis independentes ou regressoras, denominadas \(X_1, X_2, \ldots, X_k\). A relação entre essas variáveis é caracterizada por um modelo matemático chamado de equação de regressão. O modelo de regressão é ajustado a um conjunto de dados amostrais.

Modelos de regressão são frequentemente usados para analisar dados de um experimento não planejado, tal como pode surgir de observações de um fenômeno não controlado ou registros históricos.

14.1.2 Modelo de Regressão Linear Simples

Na regressão linear objetiva-se determinar relação entre uma única variável regressora \(X\) e uma variável resposta \(Y\). Pode-se assumir que a variável regressora \(X\) seja contínua e controlada pelo pesquisador. Caso o experimento seja planejado, escolhem-se os valores de \(X\) e observam-se os valores correspondentes de \(Y\).

Suponha que a verdadeira relação entre \(Y\) e \(X\) seja uma linha reta e que a observação \(Y\) para cada nível de \(X\) seja uma variável aleatória. O valor esperado de \(Y\) para cada valor de \(X\) é:

\[E(Y|X) = \beta_0 + \beta_1 X\]

em que os parâmetros \(\beta_0\) e \(\beta_1\) são constantes desconhecidas. Assume-se que cada observação \(Y\) pode ser escrita pelo modelo:

\[y = \beta_0 + \beta_1 x + e\]

Sendo \(e\) o erro aleatório com média zero e variância \(\sigma^2\), o erro \(e \sim N(0, \sigma^2)\). Os erros são variáveis aleatórias não correlacionadas com suposição de normalidade. O modelo de regressão envolve somente uma variável regressora \((x)\) e, por isso, é chamado “Modelo de Regressão Linear Simples”.

14.1.2.1 O Método dos Mínimos Quadrados

Sejam \(n\) pares de dados \((y_1, x_1), (y_2, x_2), \ldots, (y_n, x_n)\). Para estimar os parâmetros do modelo \(\beta_0\) e \(\beta_1\) pelo método dos mínimos quadrados, o qual não faz suposição sobre a distribuição dos dados, obtêm-se:

\[\hat{\beta}_1 = \frac{S_{xy}}{S_{xx}}\]

onde:

\[S_{xy} = \sum_{j=1}^{n} x_j y_j - \frac{\sum_{j=1}^{n} x_j \sum_{j=1}^{n} y_j}{n}\]

\[S_{xx} = \sum_{j=1}^{n} x_j^2 - \frac{(\sum_{j=1}^{n} x_j)^2}{n}\]

\[\hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}\]

Portanto, \(\hat{\beta}_0\) e \(\hat{\beta}_1\) são os estimadores de mínimos quadrados do intercepto e inclinação, respectivamente. O modelo de regressão linear simples ajustado é dado por:

\[\hat{y}_j = \hat{\beta}_0 + \hat{\beta}_1 x_j\]

e dá uma estimativa pontual da média de \(Y\) para cada valor de \(X\).

Estimação dos parâmetros no R:

model <- lm(y ~ x)

14.1.2.2 Forma Matricial

Na forma matricial, sejam os vetores:

\[Y = \begin{bmatrix} y_1 \\ y_2 \\ \vdots \\ y_n \end{bmatrix}, \quad X = \begin{bmatrix} 1 & x_1 \\ 1 & x_2 \\ \vdots & \vdots \\ 1 & x_n \end{bmatrix}, \quad \boldsymbol{\beta} = \begin{bmatrix} \beta_0 \\ \beta_1 \end{bmatrix}, \quad e = \begin{bmatrix} e_1 \\ e_2 \\ \vdots \\ e_n \end{bmatrix}\]

Note que o número de colunas de \(X\) é igual ao número de elementos de parâmetros a serem estimados \((\boldsymbol{\beta})\) e o número de linhas de \(X\) é o tamanho da amostra. A primeira coluna de \(X\) é um vetor com os valores que multiplicam \(\beta_0\), portanto, um vetor com elementos iguais a um (1). A segunda coluna de \(X\) é um vetor com os valores que multiplicam \(\beta_1\), portanto os valores \(x_1, x_2, \ldots, x_n\).

O vetor aleatório ‘e’ (erro) é composto de variáveis independentes, com distribuição normal \(e \sim N(0, \sigma^2 I)\). O vetor esperança dos elementos de \(e_i\) é o vetor nulo de dimensão \(n\) e matriz, cuja diagonal é formada pelas variâncias e os demais elementos são as covariâncias \(Cov[e_i, e_j] = 0\), se \(i \neq j\) com \(i, j = 1, 2, \ldots, n\).

Assim, pode-se assumir o modelo de regressão linear simples amostral da forma:

\[Y = X\boldsymbol{\beta} + e\]

O estimador de mínimos quadrados é:

\[\hat{\boldsymbol{\beta}} = (X'X)^{-1}X'Y\]

Implementação no R:

X <- as.matrix(cbind(1, x))  # Matriz X
Y <- as.matrix(cbind(y))      # Matriz Y
XtX <- t(X) %*% X             # (X'X)
XtY <- t(X) %*% Y             # X'Y
INV <- solve(XtX)             # Inversa da matriz (X'X)
Betas <- INV %*% XtY          # Coeficientes

14.1.3 Coeficiente de Correlação Linear de Pearson

Tem por objetivo medir o grau de associação entre duas variáveis. O instrumento empregado para a medida da correlação linear de Pearson, representado pela letra \(r\), é obtido por:

\[r = \frac{S_{xy}}{\sqrt{S_{xx} \times S_{yy}}}\]

onde:

\[S_{yy} = \sum y^2 - \frac{(\sum y)^2}{n}\]

\[S_{xx} = \sum x^2 - \frac{(\sum x)^2}{n}\]

\[S_{xy} = \sum xy - \frac{(\sum x)(\sum y)}{n}\]

O coeficiente de correlação é um número sem dimensão (adimensional) cujo valor se situa entre \((-1; +1)\). Quando \(X\) e \(Y\) variam no mesmo sentido, diz-se que a correlação é positiva, assim, o coeficiente de correlação tem sinal positivo. Quando \(X\) e \(Y\) variam em sentido contrário, diz-se que a correlação é negativa, assim, o coeficiente de correlação tem sinal negativo, ou seja:

  • Se \(r = 1\), a correlação é positiva perfeita
  • Se \(r = -1\), a correlação é negativa perfeita
  • Se \(r = 0\), a correlação é nula

O sinal da correlação indica qual tendência da variação conjunta das duas variáveis consideradas, entretanto, deve-se considerar também a intensidade ou o grau de correlação, onde pelo diagrama de dispersão é possível visualizar algumas tendências dessa correlação.

Diagrama de dispersão e correlação no R:

plot(y ~ x)
cor(x, y)

14.1.3.1 Teste de Hipóteses para Correlação

Para testar a hipótese de que o coeficiente de correlação seja igual a zero, as hipóteses são:

  • \(H_0: \rho = 0\)
  • \(H_1: \rho \neq 0\)

O teste estatístico apropriado para esta hipótese é:

\[t_0 = \frac{r\sqrt{n-2}}{\sqrt{1-r^2}}\]

que segue uma distribuição \(t\) com \((n-2)\) graus de liberdade, se \(H_0\) for verdadeira. Assim, rejeita-se a hipótese nula se \(|t_0| > t_{\alpha/2;(n-2)}\).

Teste de hipótese para correlação no R:

cor.test(x, y)

14.1.4 Estimação da Variância Residual

A diferença entre o valor observado \(y_j\) e o correspondente valor ajustado \(\hat{y}_j\) é denominado RESÍDUO. O \(j\)-ésimo resíduo é definido por:

\[e_j = y_j - \hat{y}_j = y_j - (\hat{\beta}_0 + \hat{\beta}_1 x_j), \quad j = 1, 2, \ldots, n\]

Os resíduos têm papel importante na verificação do ajuste do modelo e nas suposições que são realizadas.

14.1.4.1 Variância Residual da Amostra

Além de estimar \(\beta_0\) e \(\beta_1\), uma estimativa de \(\sigma^2\) (variância) é necessária para testar hipóteses e construir intervalos de confiança pertinentes ao modelo de regressão. Esta estimativa pode ser obtida dos resíduos. A soma de quadrados dos resíduos é:

\[SQRes = \sum_{j=1}^{n} e_j^2 = \sum_{j=1}^{n} (y_j - \hat{y}_j)^2\]

Substituindo \(\hat{y}_j = \hat{\beta}_0 + \hat{\beta}_1 x_j\) e após manipulação algébrica, resulta em:

\[SQRes = S_{yy} - \hat{\beta}_1 S_{xy}\]

A soma de quadrados dos resíduos tem \((n-2)\) graus de liberdade, pois dois graus de liberdade são associados com as estimativas \(\hat{\beta}_0\) e \(\hat{\beta}_1\) envolvidas na estimação de \(\hat{Y}_j\). O valor esperado da \(SQRes\) é \(E(SQRes) = (n-2)\sigma^2\), de forma que um estimador não viesado de \(\sigma^2\) é:

\[\hat{\sigma}^2 = QMRes = \frac{SQRes}{n-2}\]

Estimativa da variância residual no R:

(summary(model)$sigma)^2

14.1.4.2 Propriedades dos Estimadores de Mínimos Quadrados

Os estimadores de mínimos quadrados \((\hat{\beta}_0, \hat{\beta}_1)\) têm algumas propriedades estatísticas importantes. Note que elas são variáveis aleatórias, uma vez que são combinações lineares dos \(Y_j\) e os \(Y_j\) são variáveis aleatórias.

Esperança dos estimadores:

\[E(\hat{\beta}_1) = \beta_1\] \[E(\hat{\beta}_0) = \beta_0\]

Variância dos estimadores:

\[V(\hat{\beta}_1) = \frac{\sigma^2}{S_{xx}}\]

\[V(\hat{\beta}_0) = \sigma^2\left(\frac{1}{n} + \frac{\bar{x}^2}{S_{xx}}\right)\]

Um importante resultado relativo à qualidade dos estimadores de mínimos quadrados é o teorema de Gauss-Markov, o qual diz que para o modelo de regressão, com as suposições \(E(e) = 0\) e \(V(e) = \sigma^2\) e erros não correlacionados, os estimadores de mínimos quadrados são não-viesados e têm variância mínima quando comparado com todos outros estimadores não-viesados que são combinações lineares de \(y_j\). Esses estimadores de mínimos quadrados são chamados de BLUE (Best Linear Unbiased Estimators), em que “best” implica em variância mínima.

A distribuição dos parâmetros será:

\[\hat{\beta}_1 \sim N\left(\beta_1, \frac{\sigma^2}{S_{xx}}\right)\]

\[\hat{\beta}_0 \sim N\left(\beta_0, \sigma^2\left[\frac{1}{n} + \frac{\bar{x}^2}{S_{xx}}\right]\right)\]

14.1.5 Intervalo de Confiança para os Parâmetros

Os intervalos de confiança \((1-\alpha)100\%\) para os parâmetros \(\beta_0\) e \(\beta_1\) são dados por:

\[IC[\beta_0] = \hat{\beta}_0 \pm t_{\alpha/2;(n-2)} \sqrt{\hat{\sigma}^2\left(\frac{1}{n} + \frac{\bar{x}^2}{S_{xx}}\right)}\]

\[IC[\beta_1] = \hat{\beta}_1 \pm t_{\alpha/2;(n-2)} \sqrt{\frac{QMRes}{S_{xx}}}\]

e a distribuição \(t\) de Student com \((n-2)\) graus de liberdade.

Intervalo de confiança para os parâmetros no R:

confint(model, level = 0.95)

14.1.5.1 Intervalo de Confiança para Variância

Se os erros são normalmente e independentemente distribuídos, a distribuição amostral de \(\frac{(n-2)QMRes}{\sigma^2}\) é qui-quadrado com \((n-2)\) graus de liberdade, ou seja:

\[P\left\{\chi^2_{1-\alpha/2;(n-2)} \leq \frac{(n-2)QMRes}{\sigma^2} \leq \chi^2_{\alpha/2;(n-2)}\right\} = (1-\alpha)\]

e consequentemente:

\[IC[\sigma^2] = \left[\frac{(n-2)QMRes}{\chi^2_{\alpha/2;(n-2)}}, \frac{(n-2)QMRes}{\chi^2_{1-\alpha/2;(n-2)}}\right]\]

Intervalo de confiança para a variância no R:

gl <- n - 2
QMRes <- (summary(model)$sigma)^2
ICvar <- QMRes * (gl) / qchisq(c(0.025, 0.975), df = gl, lower.tail = FALSE)
ICvar

14.1.6 Testando Hipóteses na Regressão Linear Simples

Para testar hipóteses sobre o intercepto (coeficiente linear) \((\beta_0)\) e o coeficiente angular \((\beta_1)\) do modelo de regressão, deve-se fazer a suposição de que os \((e_j)\) são normalmente distribuídos, ou seja, assume-se que os erros \(e_j \sim NID(0, \sigma^2)\).

14.1.6.1 Teste para o Coeficiente Angular

Para testar a hipótese de que o coeficiente angular é igual a um valor constante, por exemplo, \(\beta_{1,0}\). As hipóteses apropriadas são:

  • \(H_0: \beta_1 = \beta_{1,0}\)
  • \(H_1: \beta_1 \neq \beta_{1,0}\)

em que se especificou uma hipótese alternativa bilateral. Como os \(e_j \sim NID(0, \sigma^2)\), então segue que os valores observados são \(Y_i \sim NID(\beta_0 + \beta_1 X_j, \sigma^2)\). Como \(\hat{\beta}_1\) é uma combinação linear das observações, então \(\hat{\beta}_1 \sim NID(\beta_1, \sigma^2/S_{xx})\). Além disso, \(\hat{\beta}_1\) é independente do \(QMRes\). Logo, como resultado da suposição de normalidade, a estatística:

\[t_1 = \frac{\hat{\beta}_1 - \beta_{1,0}}{\sqrt{QMRes/S_{xx}}}\]

segue uma distribuição \(t\) com \((n-2)\) graus de liberdade sob \(H_0: \beta_1 = \beta_{1,0}\). A estatística \(t_1\) é usada para testar \(H_0\) comparando-se o valor observado de \(t_1\) com o valor tabelado da distribuição \(t\): \(t_{\alpha/2;(n-2)}\). A hipótese nula será rejeitada se \(|t_1| > t_{\alpha/2;(n-2)}\).

14.1.6.2 Teste para o Coeficiente Linear

Um procedimento similar pode ser usado para testar a hipótese sobre o intercepto. Para testar \(H_0: \beta_0 = \beta_{0,0}\) versus \(H_1: \beta_0 \neq \beta_{0,0}\), usa-se a estatística teste:

\[t_0 = \frac{\hat{\beta}_0 - \beta_{0,0}}{\sqrt{QMRes\left(\frac{1}{n} + \frac{\bar{x}^2}{S_{xx}}\right)}}\]

e rejeita-se a hipótese nula se \(|t_0| > t_{\alpha/2;(n-2)}\).

Um caso especial é testar: \(H_0: \beta_1 = 0\) versus \(H_1: \beta_1 \neq 0\), cuja hipótese está relacionada com a significância da regressão. Se \(H_0: \beta_1 = 0\) não for rejeitada, isto implica que não há uma relação linear entre \(x\) e \(y\); logo o melhor estimador de \(y_j\) para qualquer valor de \(x_j\) é \(\hat{y}_j = \bar{y}\).

Testar a hipótese dos parâmetros no R:

summary(model)

14.1.7 Análise de Variância na Regressão

A determinação da equação de regressão deve ser precedida de uma análise de variância, a fim de comprovar estatisticamente se os dados apresentam a suposta relação linear entre as variáveis \(X\) e \(Y\).

Hipóteses:

  • \(H_0: \beta_1 = 0\) (não existe a regressão)
  • \(H_1: \beta_1 \neq 0\) (existe a regressão)

Tabela ANOVA:

Causa de Variação Graus de Liberdade Soma de Quadrados Quadrado Médio F
Regressão 1 \(SQReg\) \(QMReg\) \(\frac{QMReg}{QMRes}\)
Resíduo \(n-2\) \(SQRes\) \(QMRes\)
Total \(n-1\) \(SQTotal\)

Conclusão: Se \(F \geq F[1, (n-2); \alpha]\), rejeita-se \(H_0\) ao nível de significância adotado, e conclui-se que existe a regressão.

Como \(SQTotal = SQReg + SQRes = S_{yy}\):

\[SQReg = \hat{\beta}_1 S_{xy}\] \[SQRes = SQTotal - SQReg\]

Observe que ao realizar a análise de variância, o procedimento é comparar as variâncias:

\[\sum_{i=1}^{n}(y_i - \bar{y})^2 = \sum_{i=1}^{n}(y_i - \hat{y}_i)^2 + \sum_{i=1}^{n}(\hat{y}_i - \bar{y})^2\]

ou corresponde a \(SQTotal = SQRes + SQReg\).

  • \(SQTotal\) é a variação total de \(Y\) em torno da média
  • \(SQRes\) é a variação de \(Y\) em torno da reta
  • \(SQReg\) é a variação das esperanças específicas de \(Y\), em torno da média

Análise de variância na regressão no R:

anova(model)

14.1.8 Coeficiente de Determinação ou Explicação

A Soma de Quadrado Total mede a variação nas observações \(Y_i\), ou a incerteza em predizer \(Y\) quando \(X\) não é considerado. De forma análoga, Soma de Quadrado do Resíduo mede a variação em \(Y_i\) quando um modelo de regressão utilizando a variável \(X\) é empregada. Uma medida natural do efeito de \(X\) reduzindo a variação em \(Y_i\), ou seja, em reduzir a incerteza na predição de \(Y\), é expressar a redução da variação como \((SQTotal - SQRes = SQReg)\) como uma proporção da variação total:

\[R^2 = \frac{SQReg}{SQTotal} \quad \text{ou} \quad R^2 = 1 - \frac{SQRes}{SQTotal}\]

A medida \(R^2\) é chamada de coeficiente de determinação ou explicação e seu campo de variação é: \((0 \leq R^2 \leq 1)\) e indica a proporção da variação total que é “explicada” pela regressão.

  • Se \(R^2 = 1\), todos os pontos observados se situam “exatamente” sobre a reta de regressão, então as variações de \(Y\) são 100% explicadas pelas variações de \(X\) por meio da função especificada.

  • Por outro lado, um \(R^2 = 0\) pode ou não indicar ausência de correlação entre \(X\) e \(Y\).

Coeficiente de determinação no R:

summary(model)$r.squared

14.1.9 Teste de Falta de Ajuste do Modelo

Para testar estatisticamente a falta de ajuste do modelo de regressão linear simples, é necessário ter replicações, ou seja, deve-se ter pelo menos dois valores da variável resposta para alguns valores da variável regressora.

Suponha que haja \(k\) valores distintos da variável regressora \(X\): \((x_1, x_2, \ldots, x_k)\). Para o valor de \(x_i\) observa-se \(n_i\) valores da variável resposta \(y\), num total de \(n = \sum_{i=1}^{k} n_i\) observações.

Neste contexto usa-se a notação \(y_{ij}\), sendo o índice \(i\) correspondente ao índice de \(x_i\) e o índice \(j\) uma enumeração dos \(n_i\) valores:

  • \(X_1\) \((y_{11}, y_{12}, \ldots, y_{1n_1})\)
  • \(X_2\) \((y_{21}, y_{22}, \ldots, y_{2n_2})\)
  • \(\vdots\)
  • \(X_k\) \((y_{k1}, y_{k2}, \ldots, y_{kn_k})\)

A soma de quadrados dos resíduos pode ser particionada em duas somas de quadrados:

\[SQRes = SQEp + SQFa\]

onde:

  • \(SQEp\) = soma de quadrados do erro puro
  • \(SQFa\) = soma de quadrados da falta de ajuste

Hipóteses:

  • \(H_0\): O modelo de regressão linear é adequado
  • \(H_1\): O modelo de regressão linear não é adequado

A estatística teste:

\[F = \frac{SQFa/(k-2)}{SQEp/(n-k)}\]

tem distribuição \(F\) de Snedecor com \((k-2)\) e \((n-k)\) graus de liberdade. Rejeita-se a hipótese do modelo ser adequado se a estatística do teste for maior que \(F[(k-2), (n-k); \alpha]\), sendo pré-fixado o nível de significância.

Tabela ANOVA para falta de ajuste:

Fonte de Variação GL SQ QM F
Regressão 1 \(SQReg\) \(QMReg\) \(\frac{QMReg}{QMRes}\)
Resíduo \(n-2\) \(SQRes\) \(QMRes\)
Falta de ajuste \(k-2\) \(SQFa\) \(QMFa\) \(\frac{QMFa}{QMEp}\)
Erro puro \(n-k\) \(SQEp\) \(QMEp\)
Total \(n-1\) \(SQTotal\)

Falta de ajuste na regressão no R:

14.1.10 Intervalo de Confiança para \(E[Y|x]\)

A esperança de \(Y\), dados um valor de \(x\) da variável regressora, é dada por \((\beta_0 + \beta_1 x)\), sob o modelo de regressão linear. Pode-se definir um intervalo de confiança para \(E[Y|x]\), através da seguinte quantidade pivotal:

\[\frac{\hat{\beta}_0 + \hat{\beta}_1 x - (\beta_0 + \beta_1 x)}{\sqrt{\hat{\sigma}^2\left[\frac{1}{n} + \frac{(x-\bar{x})^2}{S_{xx}}\right]}}\]

que tem distribuição \(t\) de Student com \((n-2)\) graus de liberdade. Portanto, o intervalo de confiança \((1-\alpha)100\%\) para \(E[Y|x]\) é dado pela expressão:

\[\hat{\beta}_0 + \hat{\beta}_1 x \pm t_{\alpha/2;(n-2)} \sqrt{\hat{\sigma}^2\left[\frac{1}{n} + \frac{(x-\bar{x})^2}{S_{xx}}\right]}\]

A amplitude deste intervalo é:

\[2 \cdot t_{\alpha/2;(n-2)} \sqrt{\hat{\sigma}^2\left[\frac{1}{n} + \frac{(x-\bar{x})^2}{S_{xx}}\right]}\]

O intervalo de confiança fornece informação sobre a precisão das estimativas, no sentido de que quanto menor a amplitude do intervalo maior a precisão. Com os cálculos dos intervalos de confiança para alguns valores de \(x\), pode-se esboçar uma região em torno da reta estimada, indicando os limites superiores e inferiores desses intervalos. Essa região também é chamada de bandas de confiança.

Intervalo de confiança para \(E[Y|x]\) no R:

predict(model)
icconf <- predict(model, interval = "confidence")
icconf

14.1.11 Intervalo de Predição

Suponha que agora estamos interessados em fazer uma predição para um valor futuro de \(Y_0\) (não observado) correspondente a \(X = x_0\). Este intervalo às vezes é mais interessante para o cientista do que propriamente o intervalo de confiança para as respostas médias.

Para o modelo de regressão linear simples bem ajustado, diz-se que \(\hat{Y} = \hat{\beta}_0 + \hat{\beta}_1 X_j\) é uma predição de \(Y_0\). Note-se que a predição de \(Y_0\) é igual ao estimador do valor esperado de \(Y\), para \(X = x_0\). O erro de predição é definido por: \((Y_0 - \hat{Y}_0)\) o qual será a base para construirmos um intervalo de predição.

Temos \(Y_0 \sim N(\beta_0 + \beta_1 X_0, \sigma^2)\). A variável \(Y_0\) não faz parte da amostra utilizada no ajuste do modelo, portanto é independente de \(\hat{Y}_0\). A variância do erro da predição pode ser escrita como a soma das variâncias das variáveis que a compõem, dada a independência entre os termos. Os resultados acima implicam:

\[E[Y_0 - \hat{Y}_0] = 0\]

\[V[Y_0 - \hat{Y}_0] = \sigma^2\left[1 + \frac{1}{n} + \frac{(x_0-\bar{x})^2}{S_{xx}}\right]\]

Como \([Y_0 - \hat{Y}_0] \sim N\left(0, \sigma^2\left[1 + \frac{1}{n} + \frac{(x_0-\bar{x})^2}{S_{xx}}\right]\right)\)

O intervalo de predição \((1-\alpha)100\%\) de \(Y\), para \(X = x_0\) é definido por:

\[\hat{Y}_0 \pm t_{\alpha/2;(n-2)} \sqrt{\hat{\sigma}^2\left(1 + \frac{1}{n} + \frac{(x_0-\bar{x})^2}{S_{xx}}\right)}\]

A amplitude do intervalo de predição é:

\[2 \cdot t_{\alpha/2;(n-2)} \sqrt{\hat{\sigma}^2\left(1 + \frac{1}{n} + \frac{(x_0-\bar{x})^2}{S_{xx}}\right)}\]

O conceito de intervalo de previsão é análogo ao intervalo de confiança, com a diferença de que, enquanto o intervalo de confiança se refere a uma constante (o parâmetro \(\beta\)), o intervalo de previsão tem a probabilidade \((1-\alpha)\) de conter não o parâmetro populacional, mas um futuro valor de \(y_0\) da variável aleatória \(Y_0\).

As bandas de confiança bem como as bandas de predição têm um formato de hipérbole, o que enfatiza o risco de fazer extrapolações, ou seja, predições fora do intervalo observado da variável \(X\). Portanto, os modelos devem ser usados com cautela para fazer previsões sobre a variável resposta.

Intervalo de predição no R:

icpred

14.1.12 Análise de Resíduo

A análise de resíduos desempenha papel fundamental na avaliação do ajuste de um modelo de regressão linear simples, investiga a adequação do modelo quanto às suposições básicas do modelo, bem como:

  1. Independência das variáveis erros, \(e_i\)
  2. Homogeneidade das variâncias de \(e_i\)
  3. Normalidade dos erros \(e_i\), \(i = 1, 2, \ldots, n\)
  4. Relação linear das variáveis \(x\) e \(y\)

e assim, avalia-se a falta de ajuste do modelo proposto. Além dos testes de significância e adequação, a análise de resíduo vem complementar o elenco de procedimentos que devem ser realizados após o ajuste de qualquer modelo.

14.1.12.1 Propriedades de Resíduo

Considerando \(i\) o número de elementos da amostra, se o erro: \(e_i = (y - \hat{y})\) com \(i = 1, 2, \ldots, n\):

P1: \(E[e_i] = 0\)

P2: \(Var[e_i] = \sigma^2\left[1 - h_{ii}\right]\), considerando:

\[h_{ii} = \frac{1}{n} + \frac{(x_i - \bar{x})^2}{S_{xx}}\]

\[h_{ij} = -\frac{1}{n} + \frac{(x_i - \bar{x})(x_j - \bar{x})}{S_{xx}}\]

P3: \(Cov[e_i, e_j] = -\sigma^2 h_{ij}\) com \(i, j = 1, 2, \ldots, n\); \(i \neq j\)

No modelo, há a suposição de normalidade dos erros \(e_i\), tem-se que \(Y_i\) tem distribuição normal e os resíduos, os quais são uma combinação linear dos \(y\)’s, também têm distribuição normal, ou seja, \(e_i \sim N(0; \sigma^2(1-h_{ii}))\).

14.1.12.2 Tipos de Resíduos

Resíduos Padronizados são escalonados para reduzir uma variável aleatória a ter esperança com média zero e seus desvios padrão seja aproximadamente igual a um. Consequentemente \(d_i > 3\) indica outliers.

\[d_i = \frac{e_i}{\sqrt{QMRes}} = \frac{e_i}{\hat{\sigma}}\]

com \(i = 1, 2, \ldots, n\).

14.1.13 Transformações Usuais

14.1.13.1 Método Analítico – Box Cox

Muitas transformações são escolhidas empiricamente, porém técnicas formais podem ser aplicadas para apoiar a escolha de uma transformação apropriada. Uma útil classe de transformações para \(Y\) é a transformação potência \(Y^{\lambda}\) em que \(\lambda\) é o parâmetro a ser determinado. Box e Cox mostraram como \(\lambda\) pode ser estimado.

O procedimento utilizado é:

Ajustar o modelo para vários valores de \(\lambda\) obtendo a \(SQRes(\lambda)\), isto é, a \(SQRes\) para cada um dos \(\lambda\) usados. Note que as \(SQRes(\lambda)\) não são comparáveis diretamente se utilizarmos os resultados das regressões de \(Y^{\lambda}\) em \(X\) porque para cada \(\lambda\), a \(SQRes\) é medida em uma escala diferente. Para que a comparação das \(SQRes\) seja possível, deve-se utilizar para cada \(\lambda\) escolhido:

Para \(\lambda \neq 0\): \[Y^{\lambda} = \frac{y^{\lambda} - 1}{\lambda}\]

Para \(\lambda = 0\): \[Y^{\lambda} = \ln y\]

Uma variante do modelo apresentado por Box Cox é dada por:

Para \(\lambda \neq 0\): \[Y^{\lambda} = \frac{y^{\lambda} - 1}{\lambda} \cdot \frac{1}{y^{\lambda-1}}\]

Para \(\lambda = 0\): \[Y^{\lambda} = y \cdot \ln y\]

Em que \(y_\cdot = \exp\{1/n \cdot \sum \ln y_i\}\), é a média geométrica das observações.

Para calcular o intervalo de confiança para o \(\lambda\) que nos fornece a menor \(SQRes\). Este intervalo é obtido graficamente calculando-se:

\[SS^* = SQRes(\lambda)\left[1 + \frac{t^2_{\alpha/2,\nu}}{\nu}\right]\]

em que \(\nu = n - p\), sendo \(n\) o tamanho amostral e \(p\) o número de parâmetros estimados. Com o valor obtido em \(SS^*\) traçar uma reta paralela ao eixo \(\lambda\) obtendo-se limites (inferior e superior) do intervalo de confiança. Caso o intervalo de confiança obtido não contenha o valor numérico um, conclui-se que a transformação é útil.

Transformação Box Cox no R:

require(MASS)
boxcox(y ~ x)

Para encontrar o valor do lambda diretamente no R:

boxd1 <- boxcox(y ~ x, lambda = seq(-1, 1.9, length = 100))
lambda <- boxd1$x[which(boxd1$y == max(boxd1$y))]
lambda

14.2 Regressão Linear Múltipla

14.2.1 Modelo de Regressão

A regressão linear múltipla existe quando se admite que o valor da variável dependente esteja em função linear de duas ou mais variáveis independentes. Em geral, a variável dependente (resposta) \(Y\), pode estar relacionada a \(k\) variáveis independentes ou regressoras.

O modelo de regressão linear múltipla com \(k\) variáveis independentes é:

\[Y = \beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} + \cdots + \beta_k x_{ik} + e_i\]

onde \(e \sim N(0, \sigma^2)\), com \(i = 1, 2, \ldots, n\).

Os \(\beta_j\) com \(j = 1, 2, \ldots, k\), são chamados de coeficientes de regressão. Esse modelo descreve um hiperplano no espaço \(k\)-dimensional dos regressores \(X_i\). O parâmetro \(\beta_j\) representa a variação esperada na resposta \(Y\) por unidade de variação unitária em \(x_i\), quando todos os outros regressores \(x_i\) \((i \neq j)\) forem mantidos constantes.

O modelo estatístico na forma matricial:

\[Y = X\boldsymbol{\beta} + e\]

onde \(e \sim NM_n(0, \sigma^2 I)\).

Componentes:

  • \(Y\) é o vetor, de dimensão \((n \times 1)\), associado à variável resposta \(Y\)
  • \(X\) é a matriz, de dimensão \((n \times p)\), dos níveis das variáveis independentes
  • \(\boldsymbol{\beta}\) é o vetor, de dimensão \((p \times 1)\), dos parâmetros desconhecidos
  • \(e\) é o vetor dos erros aleatórios

O modelo para \(k\) variáveis regressoras \(X_1, X_2, \ldots, X_k\) usando a notação matricial é:

\[Y = \begin{bmatrix} y_1 \\ y_2 \\ \vdots \\ y_n \end{bmatrix}, \quad X = \begin{bmatrix} 1 & x_{11} & x_{12} & \cdots & x_{1k} \\ 1 & x_{21} & x_{22} & \cdots & x_{2k} \\ \vdots & \vdots & \vdots & \ddots & \vdots \\ 1 & x_{n1} & x_{n2} & \cdots & x_{nk} \end{bmatrix}, \quad \boldsymbol{\beta} = \begin{bmatrix} \beta_0 \\ \beta_1 \\ \vdots \\ \beta_k \end{bmatrix}, \quad e = \begin{bmatrix} e_1 \\ e_2 \\ \vdots \\ e_n \end{bmatrix}\]

Nota-se que:

  • A primeira coluna de \(X\) é um vetor de dimensão \(n\), cujos elementos são todos iguais a 1, correspondendo ao coeficiente \(\beta_0\) do modelo de regressão múltipla
  • O número de colunas de \(X\) é igual ao número de elementos em \(\boldsymbol{\beta}\)
  • O número de linhas de \(X\) é o tamanho da amostra
  • Cada coluna de \(X\), exceto a primeira, é um vetor de dimensão \(n\) formado pelos valores correspondentes às observações da amostra, que multiplicam o respectivo elemento de \(\boldsymbol{\beta}\), para cada regressora

De forma semelhante à regressão linear simples, tem-se os seguintes pressupostos:

  1. A variável dependente \(Y\) é função linear das variáveis regressoras \(X_j\) \((j=1,2,\ldots,k)\)
  2. As variáveis regressoras \(X_j\) são fixas (são variáveis aleatórias)
  3. \(E(e_j) = 0\), sendo 0 um vetor de zeros de dimensões \((n \times 1)\)
  4. Os erros são homocedásticos, isto é, \(Var(e) = E(e_j^2) = \sigma^2\)
  5. Os erros são independentes, ou seja, \(Cov(e_j, e_{j'}) = E(e_j, e_{j'}) = 0\), para \(j \neq j'\)
  6. Os erros têm distribuição normal

Por (iv e v) tem-se \(Var(e) = E(e_j, e_{j'}) = \sigma^2 I\) de dimensões \((n \times n)\). Assim como (vi) tem-se: \(e \sim NM_n(0, \sigma^2 I)\) e \(Y \sim NM_n(X\boldsymbol{\beta}, \sigma^2 I)\), pois a esperança e variância:

\[E[Y|x] = E[X\boldsymbol{\beta} + e] = X\boldsymbol{\beta}\] \[Var[Y|x] = Var[X\boldsymbol{\beta} + e] = \sigma^2 I\]

Estimando os parâmetros do modelo no R:

model <- lm(Y ~ X1 + X2)
model

14.2.2 Estimativas dos Parâmetros pelo Método dos Mínimos Quadrados

O método dos mínimos quadrados pode ser usado para estimar os coeficientes de regressão no modelo de regressão múltipla. Sejam \((\hat{\boldsymbol{\beta}}\) e \(\hat{e})\) os vetores das estimativas dos parâmetros e dos desvios, tem-se:

\[Y = X\hat{\boldsymbol{\beta}} + \hat{e}\]

então

\[\hat{e} = Y - X\hat{\boldsymbol{\beta}}\]

A soma de quadrado dos desvios é dada por:

\[Z = \hat{e}'\hat{e} = (Y - X\hat{\boldsymbol{\beta}})' \cdot (Y - X\hat{\boldsymbol{\beta}}) = (Y'Y - Y'X\hat{\boldsymbol{\beta}} - \hat{\boldsymbol{\beta}}'X'Y + \hat{\boldsymbol{\beta}}'X'X\hat{\boldsymbol{\beta}})\]

Como \(Y'X\hat{\boldsymbol{\beta}} = \hat{\boldsymbol{\beta}}'X'Y\) são iguais, pois uma é a transposta da outra e cada uma tem apenas um elemento, então \((Y'Y - 2\hat{\boldsymbol{\beta}}'X'Y + \hat{\boldsymbol{\beta}}'X'X\hat{\boldsymbol{\beta}})\). A função \(Z\) apresenta ponto mínimo para os valores de \(\hat{\boldsymbol{\beta}}\) que tornem a diferencial identicamente nula para \(X'X\hat{\boldsymbol{\beta}} = X'Y\) que é o sistema de equações normais.

Se \(X'X\) é não singular, existe a matriz inversa \((X'X)^{-1}\), obtém-se:

\[\hat{\boldsymbol{\beta}} = (X'X)^{-1}X'Y\]

Para obtenção dos parâmetros, a construção das matrizes: \(X'X\hat{\boldsymbol{\beta}} = X'Y\)

\[(X'X) = \begin{bmatrix} n & \sum x_{i1} & \sum x_{i2} & \cdots & \sum x_{ik} \\ \sum x_{i1} & \sum x_{i1}^2 & \sum x_{i1}x_{i2} & \cdots & \sum x_{i1}x_{ik} \\ \sum x_{i2} & \sum x_{i1}x_{i2} & \sum x_{i2}^2 & \cdots & \sum x_{i2}x_{ik} \\ \vdots & \vdots & \vdots & \ddots & \vdots \\ \sum x_{ik} & \sum x_{i1}x_{ik} & \sum x_{i2}x_{ik} & \cdots & \sum x_{ik}^2 \end{bmatrix} \begin{bmatrix} \hat{\beta}_0 \\ \hat{\beta}_1 \\ \vdots \\ \hat{\beta}_k \end{bmatrix} = \begin{bmatrix} \sum Y_i \\ \sum x_{i1}Y_i \\ \sum x_{i2}Y_i \\ \vdots \\ \sum x_{ik}Y_i \end{bmatrix}\]

Como \(p = k + 1\) equações normais, uma para cada um dos coeficientes desconhecidos de regressão. A solução para as equações normais serão os estimadores de mínimos quadrados dos coeficientes de regressão \(\hat{\beta}_0, \hat{\beta}_1, \ldots, \hat{\beta}_k\). As equações normais podem ser resolvidas por qualquer método apropriado para resolver um sistema de equações lineares.

14.2.2.1 Propriedades dos Estimadores de Mínimos Quadrados

P1) Se \(\hat{\boldsymbol{\beta}}\) é um estimador não viesado para \(\boldsymbol{\beta}\), tem-se:

\[E(\hat{\boldsymbol{\beta}}) = \boldsymbol{\beta}\]

P2) A matriz de variâncias e covariâncias de \(\hat{\boldsymbol{\beta}}\) é dada por:

\[Var(\hat{\boldsymbol{\beta}}) = \sigma^2(X'X)^{-1}\]

As variâncias dos \(\hat{\beta}\)’s são expressas em termos dos elementos da inversa da matriz \(X'X\). A matriz \(X'X\) multiplicada pela constante \(\sigma^2\) representa a matriz de covariância dos coeficientes de regressão \(\hat{\boldsymbol{\beta}}\). Os elementos da diagonal de \(\sigma^2(X'X)^{-1}\) são as variâncias dos \(\hat{\beta}_j\) e os elementos fora da diagonal são as covariâncias.

P3) Se \(C = (X'X)^{-1}\) é simétrica, tem-se:

\[Var(\hat{\beta}_j) = \sigma^2 C_{jj}\]

com \(j = 0, 1, 2, \ldots, k\)

\[Cov(\hat{\beta}_i, \hat{\beta}_j) = \sigma^2 C_{ij}\]

com \(i \neq j\)

logo, a distribuição de probabilidade de \(\hat{\boldsymbol{\beta}} \sim NM_{p+1}[\boldsymbol{\beta}; \sigma^2(X'X)^{-1}]\).

14.2.3 Estimativa da Variância Residual

As estimativas das variâncias desses coeficientes são obtidas trocando \(\sigma^2\) pela estimativa \(\hat{\sigma}^2\), que é a raiz quadrada da variância estimada do \(j\)-ésimo coeficiente de regressão denominado de erro padrão estimado de \(\hat{\beta}_j\), ou seja, \(ep(\hat{\beta}_j) = \sqrt{\hat{\sigma}^2 C_{jj}}\).

A estimativa de \(\sigma^2\) é definida em termos da soma quadrática dos resíduos:

\[SQRes = \sum_{i=1}^{n} e_i^2 = \sum_{i=1}^{n} (Y_i - \hat{Y}_i)^2\]

logo o valor esperado da quantidade \(SQRes\) é \(\sigma^2(n-p)\), um estimador não tendencioso de \(\sigma^2\), dado pela média quadrática residual ou dos erros:

\[\hat{\sigma}^2 = QMRes = \frac{SQRes}{n-p}\]

Estimativa da variância residual no R:

(summary(model)$sigma)^2

14.2.4 Análise de Variância da Regressão Linear Múltipla

Ao se realizar os testes de hipóteses dos parâmetros do modelo, verifica-se a adequação do modelo. O teste de significância da regressão é um teste para determinar se existe uma relação linear entre a variável resposta \(Y\) e um subconjunto de regressores \(\{x_1, x_2, \ldots, x_n\}\).

Hipóteses apropriadas:

  • \(H_0: \beta_1 = \beta_2 = \cdots = \beta_k = 0\)
  • \(H_1: \beta_j \neq 0\) para no mínimo um \(j\)

A soma de quadrados total é:

\[SQTotal = \sum_{i=1}^{n} Y_i^2 - \frac{(\sum Y_i)^2}{n} = Y'Y - \frac{(\sum Y_i)^2}{n}\]

A soma de quadrados de regressão:

\[SQReg = \hat{\boldsymbol{\beta}}'X'Y - \frac{(\sum Y_i)^2}{n}\]

consequentemente:

\[SQTotal = SQReg + SQRes\]

Esses resultados levam à construção do seguinte esquema de análise de variância:

Tabela ANOVA para RLM:

Fonte de Variação GL SQ QM F
Regressão \((k) = p-1\) \(\hat{\boldsymbol{\beta}}'X'Y - \frac{(\sum Y_i)^2}{n}\) \(SQReg/(p-1)\) \(\frac{QMReg}{QMRes}\)
Resíduo \(n-p\) \(Y'Y - \hat{\boldsymbol{\beta}}'X'Y\) \(SQRes/(n-p)\)
Total \(n-1\) \(Y'Y - \frac{(\sum Y_i)^2}{n}\)

Conclusão: A estatística do teste para \(H_0\) tem distribuição \(F\) com \([k; (n-p)]\) graus de liberdade, então se a estatística \(F \geq F[k, n-p; \alpha]\), rejeita-se \(H_0\) ao nível de significância adotado e conclui-se que pelo menos uma das regressoras contribui-se significativamente com o modelo de regressão.

Análise de variância do modelo no R:

anova(model)

14.2.5 Teste F Parcial

O teste F Parcial tem um papel importante na construção do modelo, pois busca o melhor conjunto de regressores para utilizar no modelo. Os parâmetros são adicionados ao modelo sequencialmente.

Considerando \(k = 2\) variáveis no modelo de regressão, o modelo pode ser escrito:

\[Y = X\boldsymbol{\beta} + e\]

ou

\[Y = X_1\boldsymbol{\beta}_1 + X_2\boldsymbol{\beta}_2 + e\]

onde \(X_1\) representa as colunas de \(X\) associadas com \(\boldsymbol{\beta}_1\), e \(X_2\) representa as colunas de \(X\) associadas com \(\boldsymbol{\beta}_2\).

Para o modelo completo (incluindo \(\boldsymbol{\beta}_1\), \(\boldsymbol{\beta}_2\)) sabe-se que \(\hat{\boldsymbol{\beta}} = (X'X)^{-1}X'Y\). Se a \(SQReg\) para todas variáveis é: \(SQReg(\boldsymbol{\beta}) = \hat{\boldsymbol{\beta}}'X'Y\) com \((p-1 = k)\) graus de liberdade e o quadrado médio do resíduo, \(QMRes = \frac{Y'Y - \hat{\boldsymbol{\beta}}'X'Y}{(n-p)}\) e \(SQReg(\boldsymbol{\beta})\) é chamada soma quadrática da regressão devido a \(\boldsymbol{\beta}\).

Para detectar a contribuição dos termos em \(\boldsymbol{\beta}_1\) para a regressão, ajuste o modelo com a \(H_0: \boldsymbol{\beta}_1 = 0\) como verdadeira. O modelo reduzido será: \(Y = X_2\boldsymbol{\beta}_2 + e\). A estimativa de mínimos quadrados de \(\boldsymbol{\beta}_2\) é \(\hat{\boldsymbol{\beta}}_2 = (X_2'X_2)^{-1}X_2'Y\) e \(SQReg(\boldsymbol{\beta}_2) = \hat{\boldsymbol{\beta}}_2'X_2'Y\) com \((p - r)\) graus de liberdade; onde \(r = 1\) quando verifica a contribuição de uma variável e \((r < k)\) quando considerar a contribuição de um subconjunto de variáveis.

A soma quadrática de regressão devido a \(\boldsymbol{\beta}_1\), dado que \(\boldsymbol{\beta}_2\) está no modelo é:

\[SQReg(\boldsymbol{\beta}_1|\boldsymbol{\beta}_2) = SQReg(\boldsymbol{\beta}) - SQReg(\boldsymbol{\beta}_2)\]

com \(r = 1\) graus de liberdade, note que esta equação é o aumento na soma quadrática de regressão extra devido a \(\boldsymbol{\beta}_1\). Logo se \(SQReg(\boldsymbol{\beta}_1|\boldsymbol{\beta}_2)\) é o aumento na soma quadrática de regressão devido à inclusão das variáveis \((x_1, x_2, \ldots, x_n)\) no modelo.

A hipótese nula de \(\boldsymbol{\beta}_1 = 0\) é testada pela estatística teste:

\[F_0 = \frac{SQReg(\boldsymbol{\beta}_1|\boldsymbol{\beta}_2)/r}{QMRes}\]

Se o valor calculado da estatística \(F_0 > F[r, n-p]\), rejeita-se \(H_0\), conclui-se que no mínimo um dos parâmetros em \(\boldsymbol{\beta}_1\) não é zero, e consequentemente, no mínimo uma das variáveis \(x_1, x_2, \ldots, x_n\) em \(X_1\) contribui significativamente para o modelo de regressão. De forma análoga pode ser utilizado o modelo reduzido \(Y = X_1\boldsymbol{\beta}_1 + e\) para encontrar a contribuição dos termos de \(\boldsymbol{\beta}_2\).

Análise de variância sequencial:

Considerando \(k = 2\) variáveis no modelo de regressão:

Fonte de Variação GL SQ QM F
Regressão 2 \(SQReg(\boldsymbol{\beta})\)
\(SQReg\boldsymbol{\beta}_1\) 1 \(SQReg(\boldsymbol{\beta}_1|\boldsymbol{\beta}_0)\) \(QMReg(\boldsymbol{\beta}_1|\boldsymbol{\beta}_0)\)
\(SQReg\boldsymbol{\beta}_2\) 1 \(SQReg(\boldsymbol{\beta}_2|\boldsymbol{\beta}_1,\boldsymbol{\beta}_0)\) \(QMReg(\boldsymbol{\beta}_2|\boldsymbol{\beta}_1,\boldsymbol{\beta}_0)\) \(\frac{QMReg(\boldsymbol{\beta}_2|\boldsymbol{\beta}_1,\boldsymbol{\beta}_0)}{\hat{\sigma}^2}\)

O valor da estatística teste \(F_0 > F_{(r,n-p;\alpha)}\) rejeita-se \(H_0\) e conclui-se que no mínimo um dos parâmetros em \(\boldsymbol{\beta}_j\) não é zero, e no mínimo uma das variáveis \(\{x_1, x_2, \ldots, x_n\}\) em \(X_j\) contribui significativamente para o modelo de regressão.

14.2.6 Coeficiente de Determinação da RLM

O coeficiente de determinação múltipla \(R^2\) é definido como:

\[R^2 = \frac{SQReg}{SQTotal} = 1 - \frac{SQRes}{SQTotal}\]

onde \((0 \leq R^2 \leq 1)\).

Esta é uma medida da quantidade de redução na variabilidade de \(Y\), obtida pelo uso dos regressores \((x_1, x_2, \ldots, x_k)\).

Observações:

  1. Um valor alto do coeficiente não implica que haja boa adequação do modelo
  2. A raiz quadrada positiva de \(R^2\) é o coeficiente de correlação múltipla entre \(Y\) e o conjunto de regressores \((x_1, x_2, \ldots, x_k)\), sendo \(R\) uma medida de associação linear de \(Y\) e \((x_1, x_2, \ldots, x_k)\). Se \(k = 1\) tem-se a correlação linear simples.

Coeficiente de determinação no R:

summary(model)$r.squared

14.2.6.1 Coeficiente de Determinação Múltiplo Ajustado

Caso seja adicionado variáveis independentes no modelo consequentemente o valor do \(R^2\) também aumenta, pois a \(SQRes\) não aumenta com acréscimo de variáveis independentes e a \(SQTotal\) sempre permanece para certo conjunto de dados. Com o acréscimo de variáveis independentes no modelo sugere-se a utilização do Coeficiente de determinação múltiplo ajustado \(R_a^2\), que ajusta \(R^2\) dividindo cada soma de quadrados por seus graus de liberdade associados.

\[R_a^2 = 1 - \frac{\frac{SQRes}{n-p}}{\frac{SQTotal}{n-1}} = 1 - \frac{(n-1)SQRes}{(n-p)SQTotal} = 1 - \frac{(n-1)}{(n-p)}(1-R^2)\]

Se os valores de \(R^2\) e \(R_a^2\) diferirem, então existe possibilidade que o modelo tenha sido superespecificado, ou seja, há termos que contribuem não significativamente para o ajuste do modelo ou foram incluídos desnecessariamente.

14.2.7 Teste para os Coeficientes Individuais de Regressão

O teste parcial ou marginal testa se os coeficientes individuais de regressão são úteis para a determinação do valor potencial de cada uma das regressoras no modelo. O modelo pode ser mais efetivo com a inclusão ou retirada de uma ou mais regressoras no modelo atual.

Para testar qualquer a significância de qualquer coeficiente de regressão:

As hipóteses:

  • \(H_0: \beta_j = 0\)
  • \(H_1: \beta_j \neq 0\)

Caso a hipótese nula for aceita, indica que a regressora \(x_j\) pode ser retirada do modelo. A estatística de teste para essa hipótese é:

\[t_0 = \frac{\hat{\beta}_j}{\sqrt{\hat{\sigma}^2 C_{jj}}}\]

onde o \(C_{jj}\) é o elemento da diagonal de \((X'X)^{-1}\) correspondente a \(\hat{\beta}_j\). O denominador é o erro padrão do coeficiente \(\hat{\beta}_j\). A hipótese nula será rejeitada se \(|t_0| > t_{\alpha/2;(n-p)}\).

14.2.8 Intervalo de Confiança para Coeficientes

Para construir estimativas de intervalos de confiança para os coeficientes de regressão \(\hat{\beta}_j\), requer que os erros \(e_i \sim N(0, \sigma^2)\) com \(i = 1, 2, \ldots, n\) e as observações \(Y_i \sim N(\beta_0 + \sum_{i=1}^{p} \beta_j X_{ij}, \sigma^2)\).

Sendo o estimador de mínimos quadrados \(\hat{\boldsymbol{\beta}}\) uma combinação linear das observações, então \(\hat{\boldsymbol{\beta}}\) é normalmente distribuído com vetor médio \(\boldsymbol{\beta}\) e matriz de variâncias e covariâncias \(\sigma^2(X'X)^{-1}\).

A estatística:

\[\frac{\hat{\beta}_j - \beta_j}{\sqrt{\hat{\sigma}^2 C_{jj}}}\]

com \(j = 1, 2, \ldots, k\) tem distribuição \(t\), com \((n-p)\) graus de liberdade, sendo \(C_{jj}\) o \(j\)-ésimo elemento da matriz \((X'X)^{-1}\) e \(\hat{\sigma}^2\) a estimativa da variância do erro.

Isso conduz ao Intervalo de confiança \(100(1-\alpha)\%\) para o coeficiente de regressão \(\beta_j\), com \(j = 1, 2, \ldots, k\) no modelo de regressão linear múltipla dado por:

\[\hat{\beta}_j \pm t_{\alpha/2;(n-p)} \sqrt{\hat{\sigma}^2 C_{jj}}\]

Intervalo de confiança no R:

confint(model)

14.2.9 Intervalo de Confiança para Resposta Média

O intervalo de confiança para a resposta média em um determinado ponto, como \((x_{01}, x_{02}, \ldots, x_{0k})\). A estimativa da resposta média nesse ponto define o vetor:

\[x_0 = \begin{bmatrix} 1 \\ x_{01} \\ \vdots \\ x_{0k} \end{bmatrix}\]

A resposta média num ponto é \(E[Y|X_0] = x_0'\boldsymbol{\beta}\) que é estimada por \(\hat{\mu}_{Y|x_0} = x_0'\hat{\boldsymbol{\beta}}\). O estimador não tendencioso, uma vez que \(E[x_0'\hat{\boldsymbol{\beta}}] = x_0'\boldsymbol{\beta} = E[Y|X_0] = \mu_{Y|x_0}\), logo a variância de \(\hat{\mu}_{Y|x_0}\) é dada por:

\[V(\hat{\mu}_{Y|x_0}) = \sigma^2 x_0'(X'X)^{-1}x_0\]

O Intervalo de confiança de \(100(1-\alpha)\%\) para \(\mu_{Y|x_0}\) pode ser construído a partir de:

\[\frac{\hat{\mu}_{Y|x_0} - \mu_{Y|x_0}}{\sqrt{\hat{\sigma}^2 x_0'(X'X)^{-1}x_0}}\]

Para o modelo de regressão múltipla, um intervalo de confiança de \(100(1-\alpha)\%\) para a resposta média no ponto \((x_{01}, x_{02}, \ldots, x_{0k})\) é:

\[\hat{\mu}_{Y|x_0} \pm t_{\alpha/2;(n-p)} \sqrt{\hat{\sigma}^2 x_0'(X'X)^{-1}x_0}\]

14.2.10 Predição de Novas Observações

Um modelo de regressão pode ser utilizado para fazer futuras observações para a variável \(Y\), correspondendo valores particulares das variáveis independentes como \((x_{01}, x_{02}, \ldots, x_{0k})\).

Se \(x_0' = [1, x_{01}, x_{02}, \ldots, x_{0k}]\), então uma estimativa da futura observação \(Y_0\) nos pontos \(x_{01}, x_{02}, \ldots, x_{0k}\) é: \(\hat{Y}_0 = x_0'\hat{\boldsymbol{\beta}}\).

O intervalo de previsão de \(100(1-\alpha)\%\) para essa futura observação é:

\[\hat{Y}_0 \pm t_{\alpha/2;(n-p)} \sqrt{\hat{\sigma}^2(1 + x_0'(X'X)^{-1}x_0)}\]

Esse intervalo de confiança expressa o erro na previsão futura da distribuição no ponto \(x_0\).

14.2.11 Fator de Inflação da Variância (VIF)

Na regressão linear múltipla existem dependências entre a variável resposta \(Y\) e os regressores \(X_j\). Muitas vezes ocorrem dependências entre os regressores \(X_j\). Se essas dependências forem fortes, diz-se que existe a multicolinearidade, que afeta nas estimativas dos coeficientes de regressão e na aplicação do modelo como um todo. Determina-se o Fator de Inflação da Variância pela inversa da matriz de correlação das regressoras \(X_j\).

A presença da multicolinearidade pode ser detectada de várias maneiras:

  1. Quanto maior for VIF mais severa será a multicolinearidade, alguns autores indicam VIF > 10.

  2. Se o teste \(F\) para significância da regressão for significante, mas os testes para coeficientes individuais de regressão não forem significativos, então a multicolinearidade está presente.

Soluções:

  • Aumentar os dados com novas observações especificamente projetadas para fragmentar as dependências lineares
  • Remover certas variáveis do modelo, porém com a desvantagem de perder informações
  • Estimar os parâmetros não pelo Método dos Mínimos Quadrados, que é sensível à multicolinearidade, mas utilizar, por exemplo, o Método da regressão corrigida (Ridge Regression)

14.2.12 Estatísticas Numéricas para Diagnósticos

A importância de detectar pontos influentes (considerados outliers, ou observações não usual no conjunto de dados) devido ao impacto que causam no ajuste do modelo e nos pressupostos. Esses diagnósticos devem ser analisados em conjunto com as análises de resíduo.

14.2.12.1 Pontos de Alavanca \(h_{ii}\) (Leverage)

A matriz \(H\), matriz Hat (ou matriz chapéu) tem importante papel no diagnóstico de pontos influentes. A matriz \(H\) determina as variâncias e covariâncias do valor estimado \((\hat{Y})\) e dos erros. Os elementos \(h_{ii}\) da matriz \(H\) devem ser escritos:

\[h_{ii} = x_i'(X'X)^{-1}x_i\]

onde \(x_i'\) é a \(i\)-ésima observação do centro no espaço \(X\).

Tradicionalmente assume-se que o valor da diagonal de \(h_{ii}\) que excede \((2p/n)\) é removido do conjunto e considerado um ponto de alavanca.

14.2.12.2 DFBetas

A estatística que indica o quanto cada coeficiente de regressão \(\hat{\beta}_j\) se modifica, em unidades de desvio padrão, se a \(i\)-ésima observação for removida, é dada por:

\[DFBeta_{j,i} = \frac{\hat{\beta}_j - \hat{\beta}_{j(i)}}{\sqrt{S_{(i)}^2 C_{jj}}}\]

com \(i = 1, 2, \ldots, n\)

sendo \(C_{jj}\) é \(j\)-ésimo elemento da diagonal da matriz \(C = (X'X)^{-1}\);

\[S_{(i)}^2 = \frac{(n-p-1)QMRes - e_i^2(1-h_{ii})}{(n-p-1)}\]

com \(h_{ii}\) é o \(i\)-ésimo componente da diagonal da matriz \(H\).

A estatística DFBetas reflete a taxa de mudança provocada em \(\hat{\boldsymbol{\beta}}\) pela retirada da observação \(i\). A observação \(i\) tem influência no \(j\)-ésimo coeficiente de regressão, caso \(|DFBeta_{j,i}| > \frac{2}{\sqrt{n}}\).

14.2.12.3 DFFits

A estatística para investigar a influência da \(i\)-ésima observação nos valores ajustados (preditos) é dada por:

\[DFFits_i = \frac{\hat{Y}_i - \hat{Y}_{(i)}}{\sqrt{S_{(i)}^2 h_{ii}}} = \left(\frac{h_{ii}}{1-h_{ii}}\right)^{1/2} r_i\]

com \(i = 1, 2, \ldots, n\)

em que \(r_i\) são os resíduos estudentizados; \(\hat{Y}_i\) é o valor predito de \(\hat{Y}_{(i)}\) sem o uso da \(i\)-ésima observação. O DFFits reflete o quanto o valor ajustado se modifica, em unidades de desvio padrão, se a \(i\)-ésima observação for removida.

Certo cuidado com as observações em que \(|DFFits_i| > 2\sqrt{\frac{p}{n}}\).

14.2.12.4 Distância Cook

Na regressão múltipla pode ocorrer de algum subconjunto de observações influenciarem por estarem longe da vizinhança onde o restante dos dados foi coletado. Consequentemente à disposição dos pontos no espaço influenciam na determinação do \(R^2\), nas estimativas dos coeficientes de regressão e na magnitude da média quadrática dos erros.

A estatística para esse diagnóstico é a medida da distância desenvolvida por Dennis R. Cook. Essa é uma medida da distância ao quadrado entre a estimativa usual de mínimos quadrados de \(\boldsymbol{\beta}\), baseada em todas observações, e a estimativa obtida quando o \(i\)-ésimo ponto for removido, como \(\hat{\boldsymbol{\beta}}_{(i)}\). A medida da distância de Cook é dada por:

\[D_i = \frac{(\hat{\boldsymbol{\beta}} - \hat{\boldsymbol{\beta}}_{(i)})'(X'X)(\hat{\boldsymbol{\beta}} - \hat{\boldsymbol{\beta}}_{(i)})}{p\hat{\sigma}^2}\]

com \(i = 1, 2, \ldots, n\)

\(D_i\) consiste no quadrado do resíduo estudentizado, e reflete quando bem o modelo se ajusta à \(i\)-ésima observação \(Y_i\), e um comportamento que mede quão longe aquele ponto está do resto dos dados. O \((h_{ii}/(1-h_{ii}))\) é uma medida de distância do \(i\)-ésimo ponto do centroíde dos \((n-1)\) pontos restantes. Quando \(D_i\) for alto, indica que o \(i\)-ésimo ponto exerce influência, se \(D_i > 1\) indica que o ponto exerce grande influência.

A medida da distância de Cook pode ser também usada pela seguinte fórmula:

\[D_i = \frac{r_i^2}{p} \cdot \frac{h_{ii}}{(1-h_{ii})}\]

com \(i = 1, 2, \ldots, n\)

14.2.12.5 Influência na Precisão da Estimação

As medidas \(D_i\), \(DFBetas_{j,i}\) e \(DFFit_i\) fornecem uma visão do efeito de cada observação nos coeficientes estimados e nos valores ajustados. Não fornecem, contudo, qualquer informação sobre a precisão geral da estimação. Para expressar o papel da \(i\)-ésima observação na precisão da estimação pode ser medida:

\[COVRATIO_i = \frac{|(X'_{(i)}X_{(i)})^{-1}S_{(i)}^2|}{|(X'X)^{-1}QMRes|} = \left(\frac{S_i^2}{QMres}\right)^p \cdot \frac{1}{1-h_{ii}}\]

\(i = 1, 2, \ldots, n\) com \(h_{ii}\) é o \(i\)-ésimo componente da diagonal da matriz \(H\).

Belsley et al. (1980) sugeriram que se: \(COVRATIO_i > 1 + 3p/n\) ou se \(COVRATIO_i < 1 - 3p/n\), então, o \(i\)-ésimo ponto deve ser um possível ponto influente. Em geral, esses pontos de corte são mais apropriados para amostras grandes.

14.2.13 Variáveis Fictícias

As variáveis fictícias podem ser utilizadas com modelos de regressão simples ou múltiplos em situações em que uma ou mais variáveis nominais (categóricas) devam ser incorporadas no modelo. Para alguma variável categórica ser introduzida no modelo, deve-se criar uma ou mais variáveis assumindo valores numéricos, representando as categorias da variável nominal, onde essas variáveis criadas são as variáveis fictícias.

Uma variável fictícia pode ser representada por uma variável nominal \((A)\) com \(k\) categorias \((A_1, A_2, \ldots, A_k)\) – definindo \((k-1)\) variáveis, \(Z_1, Z_2, \ldots, Z_{k-1}\), assumindo dois valores (0 e 1) de forma que para \(i = 1, 2, 3, \ldots, k-1\), tem-se:

  • \(Z_i = 1\) se a unidade amostral pertence à categoria \(A_i\)
  • \(Z_i = 0\) se a unidade amostral pertence à categoria \(A_j\), com \(j \neq i\)

Dessa forma a variável \(Z_i\) foi definida, se tivermos a sequência \((1,0,0,0,\ldots,0)\) a unidade amostral em questão estará classificada na categoria \(A_1\), analogamente outras sequências:

  • \((0,1,0,0,\ldots,0)\) para categoria \(A_2\)
  • \((0,0,0,\ldots,1)\) para categoria \(A_{k-1}\)
  • \((0,0,0,\ldots,0)\) para categoria \(A_k\)

Essa é uma opção dentre várias para construção de variáveis fictícias. O objetivo principal é verificar se existe diferença entre os níveis de \(A\), para isso, basta que os interceptos e inclinações sejam diferentes.

O modelo de regressão linear múltiplo considerando duas categorias para variável fictícia:

\[Y = \beta_0 + \beta_1 x + \beta_2 z + \beta_3 xz + e\]

onde:

  • \(Y\) é a variável resposta
  • \(x\) é a variável regressora
  • \(z\) é a variável categórica

Na forma matricial \(Y = X\boldsymbol{\beta} + e\):

\[X = \begin{bmatrix} 1 & x_1 & z_1 & x_1z_1 \\ 1 & x_2 & z_2 & x_2z_2 \\ \vdots & \vdots & \vdots & \vdots \\ 1 & x_n & z_n & x_nz_n \end{bmatrix} \quad e \quad \boldsymbol{\beta} = \begin{bmatrix} \beta_0 \\ \beta_1 \\ \beta_2 \\ \beta_3 \end{bmatrix}\]

Exemplo: Para verificar se existem diferenças quando são levados em conta os salários de homens e mulheres, separadamente dois modelos:

  • Para \((Z=0)\) tem-se: \(Y_1 = \beta_0 + \beta_1 x\)
  • Para \((Z=1)\) tem-se: \(Y_2 = (\beta_0 + \beta_2) + (\beta_1 + \beta_3)x\)

Estimar o vetor dos parâmetros temos: \(\hat{\boldsymbol{\beta}} = (X'X)^{-1}X'Y\).

Estimação de parâmetros com variável qualitativa no R:

modelo <- lm(Y ~ X1 + Z + X1 %in% Z)
modelo

14.2.13.1 Teste de Coincidência das Retas

O teste para verificar se um único modelo se ajusta para ambas as categorias é realizado nas seguintes hipóteses:

  • \(H_0: \beta_2 = \beta_3 = 0\)
  • \(H_1\): pelo menos um difere de zero

Usa-se a estatística F parcial:

\[F(Z,XZ|X) = \frac{(SQReg(X,Z,XZ) - SQReg(X))/2}{QMRes(X,Z,XZ)}\]

e tem distribuição \(F(r, glres)\), logo rejeita-se \(H_0\) em favor \(H_1\) se F-parcial \(> F(r, glres; \alpha)\).

14.2.13.2 Teste de Paralelismo das Retas

O teste para verificar se os dois modelos são paralelos, basta testar as hipóteses:

  • \(H_0: \beta_3 = 0\)
  • \(H_1: \beta_3 \neq 0\)

e a estatística teste:

\[T_{\beta_3} = \frac{\hat{\beta}_3}{\sqrt{QMRes \cdot C_{33}}}\]

que sob a hipótese nula, tem distribuição \(t\) de Student com \((n-p-1)\) graus de liberdade e para um teste de nível \(\alpha\), rejeitamos \(H_0\) se \(|T_{\beta_3}| > t_{\alpha/2;(glres)}\).

14.2.13.3 Teste da Igualdade dos Interceptos

O teste para verificar a igualdade dos interceptos, basta testar as seguintes hipóteses:

  • \(H_0: \beta_2 = 0\)
  • \(H_1: \beta_2 \neq 0\)

e a estatística teste:

\[T_{\beta_2} = \frac{\hat{\beta}_2}{\sqrt{QMRes \cdot C_{22}}}\]

que sob a hipótese nula, tem distribuição \(t\) de Student com \((n-p-1)\) graus de liberdade, e para um teste de nível \(\alpha\), rejeitamos \(H_0\) se \(|T_{\beta_2}| > t_{\alpha/2;(glres)}\).

14.3 Seleção de Variáveis Regressoras

Nesta abordagem o problema específico é a escolha das variáveis regressoras para serem incluídas no modelo de regressão múltipla, ou seja, devem-se incluir todas as variáveis regressoras disponíveis ou incluir apenas um subconjunto destas variáveis. Após a decisão das variáveis regressoras selecionadas, verificar a significância e a adequação do novo modelo ajustado e realizar a análise de resíduo.

Seguem os seguintes procedimentos:

  • Todas regressões possíveis
  • Método “passo a frente” (forward)
  • Método “passo atrás” (backward)
  • Método “passo a passo” (stepwise)

14.3.1 Todas as Regressões Possíveis

Essa abordagem requer o ajuste de todas as equações de regressão envolvendo uma variável candidata, todas equações de regressão envolvendo duas variáveis candidatas e assim por diante. As equações serão avaliadas de acordo com alguns critérios:

  • Coeficiente de determinação múltipla \((R^2)\)
  • Coeficiente de determinação múltipla ajustado \((R_a^2)\)
  • Quadrado médio do resíduo \((QMRes_{(p)})\)
  • Estatística \(C_p\) de Mallows

onde \(C_p\) é uma medida da média quadrática total do erro para o modelo de regressão. As equações de regressão que tenham tendenciosidade negligenciável terão valores de \(C_p\) próximos de \(p\), e aquele com tendenciosidade significativa terão valores maiores que \(p\). Baseando-se nessa estatística a “melhor” equação de regressão terá o modelo com valor mínimo de \(C_p\).

14.3.2 Método “Passo Atrás” (Backward)

O procedimento caracteriza-se por incorporar, inicialmente, todas as variáveis auxiliares em um modelo de regressão múltiplo (completo) e percorre etapas, nas quais uma variável por vez pode ser eliminada, conforme a menor correlação parcial com \(Y\); menor coeficiente de determinação ou menor diminuição no teste F parcial. Se numa dada etapa não houver eliminação de alguma variável, o processo é então interrompido e as variáveis restantes definem o modelo final.

Etapas:

Passo 1: Ajustar o modelo completo de \((k)\) variáveis e obter \(SQReg^c\), \(\hat{\sigma}^2\) e \(glres = (n-p)\)

Passo 2: Para cada uma das \(k\) variáveis do modelo completo do passo 1, considerar o modelo reduzido – retirando esta variável – e calcular \(SQReg^r\) e obter a estatística teste F parcial:

\[F = \frac{SQReg^c - SQReg^r}{QMRes^c}\]

Passo 3: Achar \(F_{min}\)

Passo 4: Seja \(F_{sai}\) o quantil especificado da distribuição \(F\) com 1 e \((n-p\) ou \(n-k-1)\) graus de liberdade:

  • Se \(F_{min} > F_{sai}\): interromper o processo e optar pelo modelo completo desta etapa
  • Se \(F_{min} < F_{sai}\): voltar ao passo 1, iniciando nova etapa em que o modelo completo tenha \((k-1)\) variáveis – dada a eliminação da variável cujo modelo seja igual a \(F_{min}\)

14.3.3 Método “Passo a Frente” (Forward)

O procedimento se caracteriza, pois inicialmente, toma-se a variável auxiliar ou a variável de maior coeficiente de correlação amostral observada com a variável resposta, \(Y\). Sucessivamente, uma variável por vez é incorporada, conforme a maior correlação parcial com \(Y\); maior coeficiente de determinação ou maior aumento significativo no teste F parcial. O procedimento está baseado no princípio de que os regressores devem ser adicionados ao modelo, um de cada vez, até que não haja mais candidatos a regressor que produzam um aumento na soma de quadrado de regressão.

Se numa etapa não houver uma inclusão, o processo é interrompido e as variáveis selecionadas até o momento definem o modelo final.

Etapas:

Passo 1: Ajustar o modelo reduzido de \(k=1\) variável e obter \(SQReg^r\)

Passo 2: Para cada variável que não pertence ao modelo passo 1, considerar o modelo completo com a adição desta variável extra e calcular \(SQReg^c\), \(\hat{\sigma}^2\) e obter o valor da estatística teste:

\[F = \frac{SQReg^c - SQReg^r}{QMRes^c}\]

Passo 3: Achar \(F_{max}\)

Passo 4: Seja o \(F_{entra}\) o quantil especificado da distribuição \(F\) com \([1\) e \((n-p)]\) graus de liberdade:

  • Se \(F_{max} > F_{entra}\): retornar ao passo 1, iniciando nova etapa onde o modelo reduzido tem \(k = (k+1)\) variáveis – dada a inclusão da variável cuja estatística teste é igual a \(F_{max}\)
  • Se \(F_{max} < F_{entra}\): interromper o processo e optar pelo modelo reduzido desta etapa

14.3.4 Método “Passo a Passo” (Stepwise)

O procedimento é uma generalização do procedimento “passo a frente”, quando após cada etapa de incorporação de uma variável, tem-se uma etapa em que uma das variáveis selecionadas pode ser descartada. O processo é finalizado quando nenhuma variável é incluída ou descartada.

Etapas:

Passo 1: Ajustar o modelo reduzido de \(k\) variáveis e obter \(SQReg^r\)

Passo 2: Para cada variável que não pertence ao modelo do passo 1, considerar o modelo completo – com adição desta variável extra – e calcular \(SQReg^c\), \(\hat{\sigma}^2\) e obter a estatística teste:

\[F = \frac{SQReg^c - SQReg^r}{QMRes^c}\]

Passo 3: Achar \(F_{max}\)

Passo 4: Seja \(F_{entra}\) o quantil especificado da distribuição \(F\) com \([1\) e \((n-p)]\) graus de liberdade:

  • Se \(F_{max} > F_{entra}\): passar ao passo 5, com modelo completo composto por \((k+1)\) variáveis – as variáveis do modelo do passo 1 e a variável cuja estatística teste é igual a \(F_{max}\)
  • Se \(F_{max} < F_{entra}\): passar ao passo 5, com modelo completo igual ao modelo do passo 1 ou encerrar o processo se no passo 8 da etapa anterior, nenhuma variável tiver sido eliminada

Passo 5: Ajustar o modelo completo de \(k\) variáveis – com \(k+1\) variáveis e obter \(SQReg^c\), \(\hat{\sigma}^2\) com \((n-p)\) g.l.

Passo 6: Para cada uma das \(k\) variáveis do modelo completo do passo 5, considerar o modelo reduzido – retirando esta variável – e calcular \(SQReg^r\) \((n-p)\) g.l. e obter a estatística teste:

\[F = \frac{SQReg^c - SQReg^r}{QMRes^c}\]

Passo 7: Achar \(F_{min}\)

Passo 8: Seja \(F_{sai}\) o quantil especificado da distribuição \(F\) com \([1\) e \((n-k-1)]\) graus de liberdade:

  • Se \(F_{min} > F_{sai}\): não eliminar nenhuma variável e voltar ao passo 1, iniciando nova etapa com modelo reduzido com \(k=m\) variáveis ou encerrar o processo se no passo 4, nenhuma variável tiver sido anexada
  • Se \(F_{min} < F_{sai}\): eliminar a variável cuja estatística teste for igual a \(F_{min}\) e voltar ao passo 1 iniciando nova etapa com modelo reduzido com \(k-1\) variáveis

Obs: Quanto aos graus de liberdade, sempre utilizar \(gl = n-p\), sendo \(p\) o número de parâmetros envolvidos no modelo que está sendo testado. Sempre observar o valor de \(p\) utilizado na estimativa da variância residual.

Seleção de variáveis no R:

require(Rcmdr)

# Método backward
stepwise(modelo, direction = "backward", criterion = "AIC")

# Método forward
stepwise(modelo, direction = "forward", criterion = "AIC")

# Método stepwise
stepwise(modelo, direction = "both", criterion = "AIC")
